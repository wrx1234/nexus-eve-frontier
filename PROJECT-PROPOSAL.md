# NEXUS — AI Assembly Manager for EVE Frontier

> **一句话定位**: 用 AI 自主管理 EVE Frontier Smart Assemblies 的 Telegram Bot - 你的 24/7 太空基地管家。

---

## 1. 项目名称

**NEXUS** — Neural EXecutive for Unified Stations

你的 Smart Assembly 不需要你盯着。NEXUS 替你看家、补给、交易、防御。

---

## 2. 问题和痛点

EVE Frontier 的 Smart Assembly 系统是文明建设的核心，但管理体验极其痛苦:

| 痛点 | 具体表现 |
|------|---------|
| **燃料焦虑** | Assembly 燃料耗尽 = 功能停摆。玩家必须定期手动检查并补充，错过一次就可能丢失关键窗口 |
| **多 Assembly 管理地狱** | 拥有 Gate + Storage + Turret + Network Node 的玩家，每天要逐个检查状态，纯人力不可扩展 |
| **交易时机错失** | Storage Assembly 的资源买卖依赖玩家在线，离线时错过最优价格 |
| **防御响应迟缓** | Turret Assembly 遭受攻击时，如果玩家不在线，无法及时调整防御策略 |
| **信息分散** | Assembly 状态、链上事件、资源价格散落在不同界面，缺少统一视图 |
| **7×24 不现实** | 人不可能全天候在线，但太空不会因为你睡觉而停止运转 |

**核心矛盾**: Smart Assembly 需要持续管理，但玩家的注意力和在线时间是有限的。

---

## 3. 解决方案概述

### 产品形态

**Telegram Bot + Sui Move 合约 + AI 决策引擎 + 链上监控**

玩家通过 Telegram 与 NEXUS 交互，AI 引擎持续监控链上状态并自主执行管理决策，Move 合约保障资金安全和操作权限。

### 核心功能

| # | 功能 | 说明 |
|---|------|------|
| 1 | **Assembly 仪表盘** | `/status` 一键查看所有 Assembly 的燃料、状态、收益汇总 |
| 2 | **智能燃料管理** | 监控燃料水位，低于阈值自动补充，支持自定义告警线 |
| 3 | **自动化交易** | 基于 AI 策略引擎，在 Storage Assembly 上自动挂单/成交，捕捉最优价格 |
| 4 | **防御预警系统** | 监听链上攻击事件，实时推送 Telegram 告警，建议防御调整方案 |
| 5 | **Gate 通行管理** | 智能设置 Gate 通行规则 - 白名单/黑名单/收费策略 |
| 6 | **收益追踪** | 每日/每周收益报告，含燃料成本、交易利润、通行费收入 |
| 7 | **操作日志** | 所有 AI 决策和执行记录上传 Walrus，不可篡改，完全可审计 |
| 8 | **自然语言指令** | "把 Gate 的通行费提高 20%" - 用自然语言管理 Assembly |

### 各层详解

#### Move 合约层: Vault → Assembly Manager

从 SuiJarvis 的 Vault 合约改造:

```
SuiJarvis Vault              →    NEXUS Assembly Manager
─────────────────                 ─────────────────────
deposit/withdraw              →    fuel_deposit / fuel_withdraw
strategy_execute              →    assembly_operate (补燃料/挂单/设规则)
risk_check                    →    safety_check (操作前风控校验)
owner_cap                     →    manager_cap (管理权限凭证)
```

核心改造点:
- **AssemblyRegistry**: 注册和跟踪多个 Assembly 的链上地址
- **FuelBudget**: 每周燃料预算上限，防止 AI 过度消耗
- **TradeLimit**: 单笔/日累计交易限额，硬性风控
- **EmergencyBrake**: 异常情况一键冻结所有自动操作

#### AI 引擎层

复用 SuiJarvis 策略引擎框架，新增 Assembly 专用策略:

- **燃料策略**: 基于消耗速率预测耗尽时间，提前补充
- **交易策略**: 趋势跟踪 + 均值回归，在 Storage Assembly 上自动交易
- **防御策略**: 监听攻击事件，根据攻击者数量/类型动态调整 Turret 参数
- **Gate 定价策略**: 根据通行流量动态调整收费

#### Bot 层

改造现有 Telegram Bot:

```
现有命令                      →    NEXUS 命令
─────────                         ──────────
/portfolio (持仓)             →    /status (Assembly 状态)
/strategy (策略设置)          →    /policy (管理策略设置)
/trade (交易)                 →    /trade (Assembly 交易)
/risk (风控)                  →    /safety (安全设置)
/logs (日志)                  →    /logs (操作日志)
新增                          →    /fuel (燃料管理)
新增                          →    /gate (Gate 设置)
新增                          →    /alert (告警配置)
```

#### 数据层

- **Sui GraphQL API**: 监听 Assembly 相关 Events，查询链上状态
- **EVE Frontier World API**: 获取 Assembly 坐标、周边态势、资源价格
- **轮询 + WebSocket**: 关键事件秒级响应

#### 存储层

- **Walrus**: 每次 AI 决策的完整上下文、执行结果、策略参数快照
- 不可篡改，支持事后审计 "AI 为什么做了这个决策"

## 5. 与黑客松的契合点

### "A Toolkit for Civilization" 主题匹配

EVE Frontier 的文明建设依赖 Smart Assembly 网络 - Gate 连接交通、Storage 管理物资、Turret 保障安全、Network Node 传递信息。这些是文明的基础设施。

**NEXUS 就是管理这些基础设施的工具箱。**

- 文明需要基础设施 → Smart Assembly 是基础设施
- 基础设施需要管理 → NEXUS 提供 AI 自动化管理
- 好的管理工具让文明更高效 → 从人工巡检到 AI 托管

### 目标分类奖

**主攻: Utility ($5,000)**
- NEXUS 是纯粹的实用工具，解决真实的玩家痛点
- 燃料管理、自动交易、防御预警 - 每个功能都直接提升游戏体验
- 不是概念 Demo，是可以真正用起来的产品

**副攻: Live Integration ($5,000)**
- 部署到 Stillness 测试网，连接真实 Assembly
- Demo 展示实际运行效果，不是 PPT

### 为什么 AI 管理 Assembly 是文明工具箱的核心

真实文明的特征是**自动化和专业分工**。原始部落每个人自己看守领地，现代城市有自动化的水电气管理系统。NEXUS 把 EVE Frontier 的基地管理从"手动巡检"升级到"AI 托管"，这本身就是文明进化的体现。
